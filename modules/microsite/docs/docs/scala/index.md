guardrail can generate Scala clients and servers for the following
frameworks:

* [`akka-http`](akka-http/)
* [`dropwizard`](dropwizard/)
* [`http4s`](http4s/)

Additionally, there is support for Jackson serialization by way of [`akka-http-jackson`](akka-http-jackson/).
